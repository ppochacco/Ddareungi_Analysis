# 🚲 따릉이 — 따릉이 수요 예측
**Seoul Public Bike Demand Prediction**

> 시간대·날씨·위치 기반으로 서울 공공자전거 따릉이의 각 대여소에서의 수요를 예측하는 웹 서비스

📄 **[포트폴리오 상세 보기](#)** *(링크 추가 예정)*

---

## 📌 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 기간 | 2024.11.26 – 2024.12.06 |
| 팀 구성 | 5인 |
| 담당 역할 | '아산병원기숙사부근' 대여소 데이터 분석 전담(100%), 웹 개발 30% |

---

## 💡 핵심 인사이트

- **대여/반납 Top 10 스테이션**이 전체 수요의 대부분을 차지 → 핵심 스테이션 집중 관리 필요
- **여름(5–9월)** 수요 집중, **겨울(14–16시 이후)** 급감 확인
- **PM10/PM2.5(미세먼지)** 가 예상과 달리 수요에 유의미한 영향 없음 → Feature에서 제외
- **기온·강수량**이 수요 예측의 핵심 Feature로 확인
- Feature Importance 분석 결과 **day(요일), time(시간대)** 가 최상위 변수
- **RandomForest 최종 채택**

→ **특정 시간대·날씨 조건에서 어느 스테이션의 따릉이 수요가 높을지 예측해 재배치 효율화**

## 📊 데이터

| 데이터 | 출처 | 기간 |
|--------|------|------|
| 따릉이 대여/반납 이력 | 서울 열린데이터 광장 | 2020 – 2024 |

## 🔍 분석 흐름

```
데이터 탐색 & 전처리
  ↓ 따릉이 대여/반납 이력 + 기상 데이터 결합
  ↓ 대여/반납 Top 10 스테이션 파악
  ↓ 계절 파생 변수 생성

Feature Engineering
  ↓ 시간대 구간화
  ↓ 요일 분류 (평일 / 주말)
  ↓ 미세먼지 Feature 중요도 검증 → 제외
  ↓ 기온·강수량 중심 Feature 구성

모델링
  ↓ RandomForest 학습
  ↓ Feature Importance 분석 (day, time 최상위)
  ↓ 실시간 기상 API 연동
  ↓ FastAPI 모델 서빙 → 웹 연동
```
